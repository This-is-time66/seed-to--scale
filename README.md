## 🖥️ Running Locally in VS Code

### Step 1 — Prerequisites
Make sure you have installed:
- **Python 3.11+** 
- **VS Code** 


### Step 2 — Clone / open the project
```bash
# If using Git
git clone <your-repo-url>
cd seed-to-scale

# OR just open the folder in VS Code:
# File → Open Folder → select seed-to-scale/
```

### Step 3 — Create a virtual environment
Open the VS Code terminal (`` Ctrl+` ``) and run:
```bash
# Create venv
python -m venv venv

# Activate it
# Windows:
venv\Scripts\activate
# macOS / Linux:
source venv/bin/activate
```

### Step 4 — Install dependencies
```bash
pip install -r requirements.txt
```

### Step 5 — Set up environment variables
Now open `.env` in VS Code and fill in your real values:
```
GROQ_API_KEY=your key
DATABASE_URL=your DATABASE_URL

```

> **Getting your keys:**
> - **GROQ_API_KEY** → https://console.groq.com/keys (free tier available)
> - **DATABASE_URL** → Use [Neon.tech](https://neon.tech) free PostgreSQL (copy the connection string)


### Step 6 — Create the database tables
Connect to your PostgreSQL database and run:
```sql
CREATE EXTENSION IF NOT EXISTS "pgcrypto";

CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name TEXT,
    email TEXT UNIQUE NOT NULL,
    password_hash TEXT NOT NULL,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE chat_history (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id) ON DELETE CASCADE,
    concept TEXT,
    research_data JSONB,
    visionary_plan JSONB,
    audit_feedback TEXT,
    iteration_count INTEGER,
    final_manifesto TEXT,
    created_at TIMESTAMPTZ DEFAULT NOW()
);
```

### Step 7 — Run the app
```bash
python app.py
```

## 🤗 Deploying to Hugging Face Spaces (Free)

### Step 1 — Create a Hugging Face account
Go to https://huggingface.co and sign up (free).

### Step 2 — Create a new Space
1. Click **New Space**
2. **Space name:** `seed-to-scale` (or any name)
3. **SDK:** Select **Docker**
4. **Visibility:** Public (required for free tier)
5. Click **Create Space**

### Step 3 — Push your code
```bash
# Install git-lfs (required by HF)
git lfs install

# Clone your new HF Space repo
git clone https://huggingface.co/spaces/YOUR_USERNAME/seed-to-scale
cd seed-to-scale

# Copy all your project files into this folder
# Then commit and push:
git add .
git commit -m "Initial deployment"
git push
```

### Step 4 — Add your Secrets (environment variables)
**Do NOT put your `.env` file on Hugging Face.** Instead:
1. Go to your Space → **Settings** tab
2. Scroll to **Repository secrets**
3. Add each variable one by one:
   - `GROQ_API_KEY` = your key
   - `DATABASE_URL` = your connection string
   

Hugging Face will inject these as environment variables when the Docker container runs.

### Step 5 — Wait for the build
HF will automatically detect your `Dockerfile`, build the image, and deploy. Watch the **Build logs** tab. It takes 2–5 minutes on first build.

Your app will be live at:
```
https://YOUR_USERNAME-seed-to-scale.hf.space
```
